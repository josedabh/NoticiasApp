## ⚙️ Instalación y Configuración

### Requisitos Previos
- Node.js >= 20.x.x

### Pasos para Ejecutar
```bash
git clone https://github.com/josedabh/NoticiasApp
cd NoticiasApp
npm install
ng serve
```
**Acceso**: [http://localhost:4200](http://localhost:4200)
